>Sign_in 登陆
>>User 用户
>>>Test 测试选择
>>>>Specific_Test 具体测试
>>>>>Results 结果

>>>Inquiry 查询

>>Administrator 管理员

>>>>>ExamineeInformation 考生信息

>>>QuestionBank 题库管理

>>>>CreateQuestionBank 新建题库
>>>>>QuestionBankInformation 题库内容增加、更改

>>Registered 注册
